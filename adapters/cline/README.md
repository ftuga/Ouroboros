# cline adapter

**status:** planned for v1.1.

cline exposes `rules` and `workflows`. ouroboros maps naturally:

- cline rules → wrap `active-rules.md` as a cline rule source
- cline workflows → run `session-start.sh` / `session-end.sh` as scheduled workflows
- learnings → still written to `CLAUDE.md` / `active-rules.md` (cline reads both)

planned for v1.1. contributions welcome.
